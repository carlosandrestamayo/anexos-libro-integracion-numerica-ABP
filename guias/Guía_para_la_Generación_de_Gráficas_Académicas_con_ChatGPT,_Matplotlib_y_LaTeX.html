🎓 Importancia de ChatGPT, Matplotlib y LaTeX en Investigación y Docencia¶

El uso de herramientas tecnológicas como ChatGPT, Matplotlib y LaTeX está transformando la forma en que los estudiantes, docentes e investigadores abordan la generación de recursos educativos y científicos. Estas tecnologías no solo permiten automatizar tareas repetitivas, sino que además facilitan la creación de contenido visual y matemático con altos estándares de calidad, lo cual es fundamental en áreas como la ingeniería, las ciencias básicas, la inteligencia artificial y la educación STEM (Hunter, 2007; Lamport, 1994; Kasneci et al., 2023).

En el ámbito académico, LaTeX es reconocido como un estándar para la redacción de documentos que contienen fórmulas matemáticas complejas, garantizando una presentación profesional y precisa (Lamport, 1994). Por su parte, Matplotlib se ha consolidado como una de las bibliotecas más poderosas y flexibles para la visualización de datos y la representación gráfica de funciones matemáticas en Python, permitiendo crear desde gráficos sencillos hasta visualizaciones complejas con propósitos académicos o investigativos (Hunter, 2007).

Finalmente, ChatGPT representa una nueva era de interacción con la inteligencia artificial, brindando asistencia en la redacción de código, generación de explicaciones, formulación de ejercicios y producción de material educativo personalizado (OpenAI, 2023). Su integración en el proceso de enseñanza-aprendizaje aporta eficiencia y creatividad, fortaleciendo las competencias digitales de estudiantes y profesores (Kasneci et al., 2023).

Esta guía propone un enfoque práctico para aprovechar estas tres herramientas de manera articulada, con el propósito de fortalecer las competencias en la generación de recursos visuales para matemáticas, ingeniería y análisis de datos, con aplicaciones en análisis numérico.

🖥️ ¿Qué es Google Colab?¶

Google Colab (abreviatura de Google Colaboratory) es una plataforma gratuita basada en la nube que permite a los usuarios escribir y ejecutar código en Python directamente desde un navegador web, sin necesidad de instalaciones locales. Ofrece un entorno muy similar a Jupyter Notebook, pero con la ventaja de integrar almacenamiento en Google Drive y acceso a recursos computacionales avanzados, como GPU y TPU, lo que facilita el desarrollo de proyectos que requieren procesamiento intensivo.

  • Permite la ejecución de código Python en la nube.
  • Facilita la enseñanza y el aprendizaje de programación, ciencia de datos, inteligencia artificial, entre otros campos.

  • Es ampliamente utilizado para proyectos colaborativos y para compartir notebooks fácilmente a través de enlaces.

Gracias a su entorno accesible, intuitivo y gratuito, Google Colab se ha convertido en una herramienta esencial para estudiantes, docentes e investigadores que trabajan con Python, especialmente en áreas relacionadas con ciencia de datos, inteligencia artificial y visualización de datos.

✅¿Qué es ChatGPT?¶

ChatGPT es una inteligencia artificial desarrollada por OpenAI que puede entender y generar lenguaje natural.
Una de sus funciones más útiles para estudiantes y profesionales es escribir código automáticamente y explicar su funcionamiento.

También puede generar:

  • Fórmulas matemáticas en LaTeX.
  • Código de Python, incluyendo librerías como Matplotlib, Numpy y muchas más.
  • Ejemplos, ejercicios y explicaciones claras.

🎯 Cómo solicitar a ChatGPT un código de Matplotlib¶

Podemos pedirle a ChatGPT algo como:

"Genera un código en Python usando Matplotlib para graficar la función ( y = x^2 ) en el intervalo [0, 10]. Agrega título, etiquetas y leyenda."

ChatGPT nos entregará un código muy similar a este:

🐍 Código generado por ChatGPT¶

import numpy as np
import matplotlib.pyplot as plt

# Datos
x = np.linspace(0, 10, 100)
y = x**2

# Crear la gráfica
plt.plot(x, y, label=r'$y = x^2$', color='blue')

# Personalizar
plt.title('Gráfica de $y = x^2$')
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.grid(True)

# Mostrar
plt.show()

- Ejes correctamente etiquetados.

📝 Cómo Formular Prompts para la Generación Automática de Gráficas¶

Un buen prompt debe incluir:

  • Qué función deseas graficar (expresa la ecuación clara y en LaTeX).
  • Rango de valores para x.
  • Si deseas sombrear un área.
  • Si deseas líneas verticales en incrementos definidos.
  • Si quieres flechas en los ejes.
  • Si quieres etiquetas matemáticas en LaTeX.

Ejemplo de Prompt para ChatGPT:

"Genera código Python usando Matplotlib para graficar ( f(x) = \frac{1}{\sqrt{x^2 + 4}} ) desde 0 hasta 2.5, con área sombreada de 0 a 2, líneas verticales cada 0.25, flechas en los ejes, y etiquetas en LaTeX."

📊 Ejercicio: Área Bajo la Curva $ f(x) = \cos(x) $¶

En este ejercicio se busca representar gráficamente el área bajo la curva de la función: $$ f(x) = \cos(x) $$ en el intervalo de $x = 0$ a $x = \frac{\pi}{2}$, incluyendo:

✅ Área sombreada en rojo.
✅ Líneas verticales en incrementos de $\frac{\pi}{8}$.
✅ Curva completa hasta $x = \pi$.
✅ Flechas en los ejes para representar prolongación.
✅ Etiquetas en notación LaTeX.

📝 Prompt utilizado en ChatGPT para generar el código de este ejercicio:¶

"Genera código en Python usando Matplotlib para graficar la función coseno desde 0 hasta pi, con área sombreada en rojo desde 0 hasta pi/2, incluyendo líneas verticales en incrementos de pi/8, flechas en los ejes para simular coordenadas prolongadas y etiquetas en LaTeX."

Este prompt fue formulado para obtener un código limpio, claro y adecuado para materiales educativos de análisis numérico.

In [ ]:
import numpy as np
import matplotlib.pyplot as plt

# Datos para la curva completa
x = np.linspace(0, np.pi, 300)
y = np.cos(x)

fig, ax = plt.subplots(figsize=(6, 4))

# Curva completa
ax.plot(x, y, color='blue', linewidth=2, label=r'$f(x) = \cos(x)$')

# Área sombreada de 0 a pi/2
x_fill = np.linspace(0, np.pi/2, 200)
y_fill = np.cos(x_fill)
ax.fill_between(x_fill, y_fill, color='red', alpha=0.3)

# Líneas verticales en incrementos de pi/8
x_points = np.arange(0, np.pi/2 + 0.01, np.pi/8)
for x0 in x_points:
    y0 = np.cos(x0)
    ax.plot([x0, x0], [0, y0], linestyle='--', color='black', linewidth=1)

# Flechas para ejes personalizados
xmax, ymax = np.pi + 0.2, 1.1
ax.annotate('', xy=(xmax, 0), xytext=(0, 0),
            arrowprops=dict(arrowstyle='->', linewidth=1.5, color='black'))
ax.annotate('', xy=(0, ymax), xytext=(0, 0),
            arrowprops=dict(arrowstyle='->', linewidth=1.5, color='black'))

# Configuración de etiquetas
ax.set_xticks([0, np.pi/4, np.pi/2, 3*np.pi/4, np.pi])
ax.set_xticklabels([r'$0$', r'$\frac{\pi}{4}$', r'$\frac{\pi}{2}$', r'$\frac{3\pi}{4}$', r'$\pi$'])
ax.set_yticks([0, 0.5, 1])

ax.set_xlabel(r'$x$', fontsize=12, loc='right')
ax.set_ylabel(r'$f(x)$', fontsize=12, loc='top')

ax.set_xlim(0, xmax)
ax.set_ylim(0, ymax)
ax.grid(False)

# Eliminar todos los spines (bordes del recuadro)
for spine in ['top', 'right', 'left', 'bottom']:
    ax.spines[spine].set_visible(False)

plt.legend()

# Guardar sin márgenes al exportar
#plt.savefig('coseno_area_sombreada.png', dpi=300, bbox_inches='tight', transparent=True)

plt.tight_layout()
plt.show()

📊 Ejercicio: Representación Gráfica del Área Bajo la Curva $ f(x) = \frac{1}{\sqrt{x^2 + 4}} $¶

Este ejercicio tiene como propósito representar gráficamente el área bajo la curva de la función $$f(x) = \frac{1}{\sqrt{x^2 + 4}}$$ en el intervalo de $x = 0$ a $x = 2$, incluyendo:

  • El área sombreada en rojo.
  • Líneas verticales en incrementos de $0.25$.
  • La curva completa hasta $x = 2.5$.
  • Flechas en ambos ejes para representar prolongación de los ejes coordenados.
  • Etiquetas matemáticas utilizando notación LaTeX.

📝 Prompt utilizado en ChatGPT para generar el código de este ejercicio:¶

"Genera código Python usando Matplotlib para graficar la función f(x) = \frac{1}{\sqrt{x^2 + 4}} desde 0 hasta 2.5 , con área sombreada en rojo desde 0 hasta 2, incluyendo líneas verticales en incrementos de 0.25 , flechas en los ejes para simular los ejes coordenados prolongados, y etiquetas matemáticas en LaTeX."

Este prompt fue formulado para obtener un código limpio, claro y adecuado para materiales educativos de análisis numérico.

In [ ]:
import numpy as np
import matplotlib.pyplot as plt
import os

# Datos para la curva completa
x = np.linspace(0, 2.5, 300)
y = 1 / np.sqrt(x**2 + 4)

fig, ax = plt.subplots(figsize=(6, 4))

# Curva completa
ax.plot(x, y, color='blue', linewidth=2, label=r'$f(x) = \frac{1}{\sqrt{x^2 + 4}}$')

# Área sombreada de 0 a 2
x_fill = np.linspace(0, 2, 300)
y_fill = 1 / np.sqrt(x_fill**2 + 4)
ax.fill_between(x_fill, y_fill, color='red', alpha=0.3)

# Líneas verticales en incrementos de 0.25
x_points = np.arange(0, 2.01, 0.25)
for x0 in x_points:
    y0 = 1 / np.sqrt(x0**2 + 4)
    ax.plot([x0, x0], [0, y0], linestyle='--', color='black', linewidth=1)

# Ejes personalizados con flechas
xmax, ymax = 2.8, 0.6
ax.annotate('', xy=(xmax, 0), xytext=(0, 0),
            arrowprops=dict(arrowstyle='->', linewidth=1.5, color='black'))
ax.annotate('', xy=(0, ymax), xytext=(0, 0),
            arrowprops=dict(arrowstyle='->', linewidth=1.5, color='black'))

# Configuración de etiquetas
ax.set_xticks(np.arange(0, 2.01, 0.25))
ax.set_yticks([])

ax.set_xlabel(r'$x$', fontsize=12, loc='right')
ax.set_ylabel(r'$f(x)$', fontsize=12, loc='top')
ax.set_xlim(0, xmax)
ax.set_ylim(0, ymax)
ax.grid(False)

# ✅ Eliminar el cuadro exterior (spines)
for spine in ['top', 'right', 'left', 'bottom']:
    ax.spines[spine].set_visible(False)

plt.tight_layout()
plt.show()

💾 ¿Cómo guardar una imagen .png en Matplotlib dentro de una carpeta?¶

Cuando generamos una gráfica en Matplotlib, podemos guardar la imagen dentro de una carpeta específica para organizar mejor nuestros archivos.

📌 Pasos:¶

1️⃣ Crea la carpeta (si no existe) utilizando Python.
2️⃣ Guarda la imagen dentro de esa carpeta con la función plt.savefig().

🐍 Ejemplo completo: Guardar en una carpeta llamada imagenes¶

In [ ]:
import os
import numpy as np
import matplotlib.pyplot as plt

# Crear carpeta 'imagenes' si no existe
folder = 'imagenes'
os.makedirs(folder, exist_ok=True)

# Datos para la gráfica
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Crear la gráfica
plt.plot(x, y)
plt.title('Ejemplo de gráfico guardado en carpeta')

# Definir la ruta completa del archivo
filename = os.path.join(folder, 'grafico_seno.png')

# Guardar la imagen como archivo .png dentro de la carpeta
plt.savefig(filename, dpi=300, bbox_inches='tight')

# Mostrar la gráfica
plt.show()

📂 ¿Dónde se guarda el archivo?¶

El archivo se guarda dentro de la carpeta imagenes que se crea en el mismo directorio donde ejecutas tu código.
En Google Colab, esta carpeta aparecerá en el explorador de archivos que se encuentra en la parte izquierda de la pantalla.
Desde allí podrás visualizarla, abrirla o descargar los archivos generados directamente.

Estas herramientas no solo permiten optimizar el tiempo de creación de materiales didácticos, sino que también promueven la autonomía del estudiante y su capacidad para resolver problemas mediante el uso de tecnología.

A través de esta guía, se ha demostrado cómo es posible automatizar la generación de recursos académicos de alta calidad, tales como gráficos matemáticos, representaciones de áreas bajo la curva y fórmulas en notación profesional, aplicando buenas prácticas de visualización y escritura técnica.

Además, la integración de estas herramientas en el aula contribuye al desarrollo de habilidades alineadas con las necesidades del siglo XXI, tales como el pensamiento computacional, la alfabetización en datos, el manejo de lenguajes de programación y la comunicación clara de ideas matemáticas.

Finalmente, se evidencia que el uso responsable e intencionado de la inteligencia artificial en entornos educativos no busca reemplazar el conocimiento humano, sino potenciarlo, facilitando procesos de enseñanza-aprendizaje más creativos, dinámicos y eficaces.
