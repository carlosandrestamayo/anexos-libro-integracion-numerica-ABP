Introduccin a la Regresin Logstica

La Regresin Logstica es un modelo supervisado ampliamente utilizado para problemas de clasificacin binaria.
Predice la probabilidad de pertenencia a una clase mediante la funcin logstica (sigmoide), transformando los resultados a un rango entre 0 y 1.

Caractersticas Principales:

  • Modelo interpretable y rpido de entrenar.
  • til cuando las clases son aproximadamente linealmente separables.
  • Proporciona probabilidades asociadas a las clases.
  • Permite ajustar el umbral de decisin para priorizar Recall o Precision segn el problema.

Representacin Grfica de la Regresin Logstica

In[11]:
import matplotlib.pyplot as plt
import numpy as np

# Datos simulados para representar clasificacin binaria con Regresin Logstica
np.random.seed(0)
X = np.random.randn(100)
y = (X > 0).astype(int)

# Lnea sigmoide para representar Regresin Logstica
x_values = np.linspace(-3, 3, 300)
sigmoid = 1 / (1 + np.exp(-x_values))

# Crear grfico
plt.figure(figsize=(8, 5))
plt.scatter(X, y, color='purple', label='Datos simulados (0 = Clase negativa, 1 = Clase positiva)')
plt.plot(x_values, sigmoid, color='black', linewidth=2, label='Funcin sigmoide (modelo de Regresin Logstica)')
plt.axhline(0.5, color='grey', linestyle='--', linewidth=1)
plt.title("Representacin Grfica de la Regresin Logstica")
plt.xlabel("Caracterstica X (simulada)")
plt.ylabel("Probabilidad estimada de Clase 1")
plt.legend()
plt.grid(True)
plt.show()

Conversin de etiquetas a variables numricas

Hemos transformado las etiquetas "ham" y "spam" en valores numricos para que puedan ser interpretados correctamente por el modelo de clasificacin.

La codificacin utilizada es la siguiente:

  • 0: Mensaje legtimo (ham)
  • 1: Mensaje no deseado (spam)

Esta transformacin es un paso fundamental en cualquier tarea de Machine Learning supervisado, ya que los algoritmos requieren variables numricas para realizar clculos.

Visualizacin de la Matriz de Confusin

La Matriz de Confusin es una herramienta fundamental para evaluar el desempeo de modelos de clasificacin binaria como la Regresin Logstica.

En esta representacin:

  • Las celdas de la diagonal principal corresponden a las predicciones correctas (ham como ham, spam como spam).
  • Las celdas fuera de la diagonal representan errores de clasificacin.

La visualizacin mediante seaborn.heatmap facilita la interpretacin rpida de los aciertos y errores, reforzando el anlisis ms all de las mtr
